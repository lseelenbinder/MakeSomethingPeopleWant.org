1. To block multiple sites, add one per line.
  2. Do not type the http://
  3. If you want to block the entire site, leave off the www (i.e. cnn.com instead of www.cnn.com)
  4. You can use a preceding asterisk as a wildcard (i.e. *.com blocks all .com sites, or *foo blocks all domains with the word "foo" in them)

E.g.,

  • facebook.com - blocks everything on facebook.com
  • reader.google.com - only blocks reader.google.com and not mail.google.com or www.google.com
